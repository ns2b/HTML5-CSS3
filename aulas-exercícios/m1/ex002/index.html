Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags <p>...</p>! ("comentário secreto no código, da uma checada lá :p")

Se precisar quebrar o texto em algum lugar em específico
como este, você pode usar a tag <br>!

Caso queira renderizar (imprimir) uma tag exatamente como ela é escrita, você precisará escrever
"E comercial (&) + LT;"" para o sinal de "menor que": < e "E comercial + GT;" para o sinal de "maior que": >

Bora adicionar uns símbolos especiais: ® © ™ € £ ¥ ¢ $ δ Δ ↑ ↓ → ←

Bora adicionar uns emotes retardados: 😈 (fiquei com preguiça de pegar outros kkk)
Basicamente você vai digitar "E comercial (&) + #x" seguindo com o código hexadecimal do emoji que tu escolheu, sacou?
Tu pode achar esses códigos hexadecimais em vários bancos de dados para emojis, como por exemplo no site emojipedia
